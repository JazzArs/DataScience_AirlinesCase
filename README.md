# DataScience_AirlinesCase
В рамках данного проекта была проведена обработка и анализ данных, полученных от различных источников, таких как сайты бронирования билетов, форумы частых пассажиров и агентства по продаже билетов. Проект направлен на выявление потенциально подозрительных пассажиров различных авиакомпаний в соответствии с разработанными гипотезами.
