# DataScience_AirlinesCase
В рамках данного проекта была проведена обработка и анализ данных, полученных от различных источников, таких как сайты бронирования билетов, форумы частых пассажиров и агентства по продаже билетов. Проект направлен на выявление потенциально подозрительных пассажиров различных авиакомпаний в соответствии с разработанными гипотезами.

Основной задачей было провести слияние и очистку данных, полученных из разных источников, а также анализировать поведение пассажиров, чтобы выявить аномалии и возможные связи с подозрительной деятельностью. Для этого были использованы методы обработки и анализа данных с применением статистических моделей.

В рамках проекта были выполнены следующие задачи:
- Слияние данных из различных источников (бронирование билетов, форумы и агентства).
- Очистка данных от ошибок и пропусков.
- Анализ путешествий и выделение повторяющихся паттернов поведения.
- Разработка моделей для выявления аномальных пассажиров.

Проект включает Python-скрипты, ноутбуки Jupyter для анализа, промежуточные результаты и презентационные материалы, которые могут быть использованы для дальнейшего исследования и применения на практике.

Проект состоит из 4 этапов:

1. Первичная обработка данных. Перевод всех данных к единому формату (все к единой шапке, переход к английскому языку, один часовой пояс).
2. Вторичная обработка данных. Более тщательная и тонкая чистка данных, заполнение пропусков и т.д. Анализ данных.
3. Составление гипотез и их реализация.
4. Визуализация данных, представление итогов в виде двух презентаций. Одна ориентирована на аналитиков, другая — на заказчиков, то есть людей, не нуждающихся в тонкостях реализации и которым нужен результат.
